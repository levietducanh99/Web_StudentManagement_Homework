# Student Server

Express + Mongoose server for Student Manager.

Environment:
- Set MONGO_URI to point to your MongoDB instance (default: mongodb://localhost:27017/studentdb)
- Start: `npm start` (or `npm run dev` with nodemon)

APIs:
- GET /api/students
- GET /api/students/:id
- POST /api/students
- PUT /api/students/:id
- DELETE /api/students/:id
